# 简介
通过“对接”资源网站，实现影视搜索，和打开m3u8，免去网站的色情、赌博广告(不含插片)
# 使用
在一个无跨域限制的编辑器里使用，如WebIDE、WebCat等(注：编辑器与本项目无关，可能因编辑器环境导致效果有所差异)
# 数据来源
来自一些资源网站
# 实现思路
简单粗暴的获取网站前端html，接着正则解析
# 如何拓展
*以下是对文件的简单说明，而不是教你怎么改，因为我是用AI写的*
首页(ShouYe.html)：数据来自搜狗
搜索(SouSuo.html)：来自多个资源网站，通过正则解析html实现搜索
选集(XuanJi.html)：与搜索进行联动
AnNiu.js：右下角按钮，实现收藏管理
